Code of Road

Une petite application en ligne de commande pour noter le nombre de fautes au code de la route.

Je sais que Code of Road ça existe pas en anglais, mais c'est stylé comme nom.

Télécharger Version 1.0.0


Liste des commandes

Commande Description Exemple
-a n Permet d'enregistrer une série avec "n" fautes au total à la fin de la série. cor -a 5
-s Affiche quelques statistiques (juste des moyennes pour l'instant mais ça va s'étoffer à l'avenir). cor -s
-d Affiche la liste de toutes les séries enregistrées avec le nombre de fautes réalisées pour chaque série. cor -d
-l n Affiche les "n" dernières séries enregistrées cor -l 3
